metro-rio-xml-json wants to provide information about the Metro Rio system in a
machine readable format, on XML and on JSON.

Nowadays the XML schema is as simple as that:

<stations>
    <station>
    	<code<ABC</code>
    	<name<ABCD Station</name>
    </station>
</stations>

but the idea is to be very specific, including: ways, lines, accessibility
(elevators and escalators), stores, subway entrances etc.
Feel free to colaborate with the XML schema itself or by adding data by
forking the project or sending an email.
